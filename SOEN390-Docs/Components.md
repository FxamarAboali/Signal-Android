# Potential Components List

System components will be based on critical user interface items and system components that they are involved in such interaction to provide specific functionalities.

Critical User interface components:
  - registration
    - country selection
  - preferences
    - mms preferences
    - notification preferences
    - privacy preferences
    - appearance preferences
    - chat/media preferences
    - linked device preferences
    - profile preferences
  - conversations
    - web rtc call
    - import/export sms db
    - invite friends
    - create group
    - create conversation
  - conversation
    - share attachment
    - media preview
    - message info
    - group conversation

Critical System components:
  - database
  - encryption
  - authorization manager
  - shared services
  - contact manager
  - message manager
  - attachment manager
  - push manager
